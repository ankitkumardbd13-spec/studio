# Maharana Pratap ITI Saharanpur - Connect Portal

Official management and student portal for Maharana Pratap ITI, Rankhandi, Saharanpur.

## Deployment Steps (To fix the "Welcome" page error)
1. **Delete the default file**: In your project folder, navigate to the `public/` directory and delete `index.html`. This is a placeholder created by Firebase.
2. **Build the Project**: Run `npm run build`. This will create a folder named `out`.
3. **Deploy**: Run `firebase deploy`.
